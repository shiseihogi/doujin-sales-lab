同人売上管理で詰まりやすいポイント

「売上はあるのに手元に残らない」「振込額と思ってた額が違う」と感じやすい理由をまとめました。

① 価格帯ごとに取り分が変わる

DLsite / FANZA は同じ作品でも価格帯で卸価格(=取り分)が変わります。¥1,099 と ¥1,100 で取り分が変わる「境界」があり、価格設定で月の手取りが大きく変動します。

② BOOTHは固定手数料 + 小数点切り上げ

BOOTHのサービス利用料は 5.6% + ¥45/件(2025/10/28改定後)で、1件ごとに小数点以下切り上げです。低価格帯だと固定¥45の比率が大きくなり、利益率が下がりやすい構造です。本サイトでは、月販売数を1件ずつの注文として概算しています(複数個まとめ買い・BOOST・倉庫発送・送料等は別途条件で変動)。

③ 専売 / 併売で取り分が違う

DLsite は専売・併売で取り分が変わるのが一般的(暫定値・要公式確認)。専売の方が取り分は高い傾向ですが、その分「他プラットフォームに同じ作品を出せない」露出機会の損失とのトレードオフです。

④ キャンペーン割引で手取りが目減りする

DLsite / FANZA のキャンペーン参加は集客には強い一方、平均で15%前後の値引きが入ります。年間の月販売数のうち何%がキャンペーン期間か、で実効手取り率が変わります。

⑤ インボイス未登録時の取り扱い

プラットフォームごとに対応が異なり、経過措置・段階的減額が入る場合があります。最新情報・経過措置率は DLsite / FANZA同人 の公式案内および税理士へご確認ください。

⑥ 確定申告 / 消費税課税対象判定

年間売上が一定額に達すると課税対象判定が必要です。本サイトは税務代行を行うものではありません。最終判断は税理士または所轄税務署にご相談ください。

    よくある質問

    配布Excel・シミュレータ・公式値の扱いについて。

    Excelテンプレは本当に無料ですか?

    はい、ダウンロード時点でメール登録も支払いも不要です。検証企画として無料配布しており、使用感のフィードバックを優先的に集めるために公開しています。

    BOOTH の手数料は公式値で合っていますか?

    BOOTHは公式が公開している 5.6% + ¥45/件(2025/10/28改定)を反映しています。サービス利用料は1件ごとに 小数点以下を切り上げ です。例として ¥1,100 × 100部の場合:
    1件あたり手数料 = ROUNDUP(1100 × 5.6% + 45) = ROUNDUP(106.6) = ¥107
    1件あたり手取り = 1,100 − 107 = ¥993 / 100部 = ¥99,300 となります。
    ※ 月販売数を1件ずつの注文として概算しています。複数個まとめ買い・BOOST・倉庫発送・送料等は別途条件により変動します。

    DLsite / FANZA の取り分は実数ですか?

    いいえ、本サイトの DLsite / FANZA の取り分は 2026年4月時点の暫定値を使用しています。 実数は各サークル管理画面の卸価格・取り分でご確認ください。Excelテンプレは管理画面の値を入力すれば実数で計算できる形に作っています。

    インボイス未登録時の取り扱いは正確ですか?

    各プラットフォームで取り扱いが異なり、経過措置・段階的減額が入る場合があります。本シミュレータの値は2026年4月時点の暫定値で、プラットフォーム・登録状況・時期により取り扱いは変動します。最新の取り扱いは DLsite / FANZA同人 公式案内および税理士へご確認ください。

    確定申告について教えてもらえますか?

    本サイトは税務代行・税務相談を行うものではありません。確定申告・消費税課税対象判定・インボイス登録判断などは 税理士または所轄税務署 にご相談ください。

    値が古くなった場合は更新されますか?

    手数料・卸価格に大きな変動があった場合は、Excel・シミュレータともにバージョンを上げて更新する予定です。最新版は本サイトでご確認ください。
